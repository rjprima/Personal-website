Portfolio Website

A personal portfolio website for displaying projects, links, and professional information

Status: Online

Link: [rileyprimavera.vercel.app](rileyprimavera.vercel.app)


Goal:

Create an easily maintainable personal website for displaying career information


Features:

-Project section with links

-Personal links

-Contact information/page

-Embedded resume

-dark mode


Roadmap:

- [ ] mobile-first frontend refactor


Frameworks & tools:

-Sanity.io for content management

-Next.js for frontend

-Tailwind CSS for styling

-Vercel for deployment and CI/CD


Requirements:

-node.js 18+

-git


Installation:

git clone https://github.com/rjprima/Personal-website.git

cd Personal-website

npm install

npm run build

npm run start
